API Documentation - Clock & Order Management

שעונים (Clocks)

CRUD כתובת פרמטרים Body (שדות) מה עושה? שגיאות אפשריות
GET api/clocks - - שליפת כל קטלוג השעונים -
GET api/clocks/:id id - שליפת פרטי שעון ספציפי ID לא תקין / לא נמצא
POST api/clocks - name, brand, price, stock, description, imagePath הוספת שעון חדש למלאי חסרים פרטי חובה
PUT api/clocks/:id id name, brand, price, stock, description, imagePath עדכון פרטי שעון קיים ID לא תקין / לא נמצא
DELETE api/clocks/:id id - מחיקת שעון מהמערכת ID לא תקין / לא נמצא

משתמשים (Users)

CRUD כתובת פרמטרים Body (שדות) מה עושה? שגיאות אפשריות
POST api/users/login - userName, password התחברות למערכת שם משתמש או סיסמה שגויים
POST api/users/register - userName, password, email, role רישום משתמש חדש משתמש כבר קיים / חסרים פרטים
PUT api/users/password/:id id password עדכון סיסמת משתמש ID לא נמצא / לא נשלחה סיסמה

הזמנות (Orders)

CRUD כתובת פרמטרים Body (שדות) מה עושה? שגיאות אפשריות
POST api/orders - userId, items: [{ clockId, quantity }] יצירת הזמנה חדשה מלאי חסר / חסרים פרטי משתמש
GET api/orders/user/:userId userId - צפייה בהיסטוריית הזמנות של משתמש משתמש לא נמצא
PUT api/orders/:id/status id status עדכון סטטוס הזמנה (שולם/נשלח) ID הזמנה לא תקין
